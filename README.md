## ✨ Overview

Welcome to my **Desert Journey Portfolio**. This is an immersive, cinematic web experience that takes you through a journey of identity, creation, and connection. As an AI/ML Engineer and Physics enthusiast, I've designed this space to reflect the intersection of technology and the natural world.

## 🌟 Key Features

- **🌅 Dynamic Day/Night Cycle**: A fully interactive theme system that transitions the entire desert landscape (Sky, Dunes, and Elements) between day and night.
- **🎨 Parallax Depth**: Multi-layered background and foreground elements that create a sense of depth as you explore.
- **🛠️ Interactive Elements**: Clickable campfire and lanterns that toggle the global theme, complete with localized glow effects.
- **✨ Animated Typography**: Custom SVG-driven text animations and GSAP-powered transitions.
- **📱 Responsive Glassmorphism**: A modern UI design that stays legible and beautiful across all devices.

## 🚀 Tech Stack

- **Frontend**: [React.js](https://reactjs.org/) with [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Animations**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform) & CSS Keyframes
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment**: [GitHub Pages](https://pages.github.com/)

## 📂 Project Structure

- `src/components`: UI components (Hero, About, Projects, Blog, Contact)
- `src/assets`: Optimized imagery for day and night scenes
- `dist`: Production-ready build deployed to this repository

## 🛠️ Local Development

If you want to run this project locally, head over to the [MyPortfolio](https://github.com/MohamedAli1937/MyPortfolio) source repository and follow these steps:

1. **Clone the source**:
   ```bash
   git clone https://github.com/MohamedAli1937/MyPortfolio.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run dev server**:
   ```bash
   npm run dev
   ```
